# 개발 노트

## 설치 라이브러리

- yarn

  npm install --global yarn
  yarn --version //1.22.19

- 엑시오스

  yarn add axios

- 스타일드컴포넌트

  yarn add styled-components

- 스위트얼럿

  yarn add sweetalert2
  yarn add sweetalert2-react-content

- scss

  yarn add node-sass

- 라우터

  yarn add react-router-dom

- mobX
  yarn add mobx-react
